tlsprobe - a TLS testing utility

What is tlsprobe?

tlsprobe is a simple program that probes a server for TLS Cipher Suites support.
tlsprobe is Free Software, released under the Apache License, Version 2.0.

How does it work?

It just sends TLS ClientHello messages, and, by looking at server's reply (if any), determines whether the selected Cipher Suite(s) is supported or not.
tlsprobe builds TLS messages in a "byte-by-byte" fashion and so does not need any TLS library (i.e.: no OpenSSL, no GnuTLS, no libNSS).
The program supports both single Cipher Suite support check and a full-scan mode (in which it checks for support of all the cipher suites listed in the IANA TLS Cipher Suite Registry).

Here follows some usage examples.

List all Cipher Suites supported by www.google.com (https, tcp port 443):

tlsprobe -F www.google.com

Tests if www.google.com (https, tcp port 443) supports cipher suite TLS_RSA_WITH_AES_128_CBC_SHA:

tlsprobe -c TLS_RSA_WITH_AES_128_CBC_SHA www.google.com

List all Cipher Suites supported by Active Directory Domain Controller 10.0.0.5 with LDAP over ssl/tls (ldaps, tcp port 636):

tlsprobe -F -p 636 10.0.0.5

For more help about possible options, try:

tlsprobe --help
